==================
   Intel 8051AH
(family of MCS-51)
==================

Features:
- 4096 bytes of on-chip program memory.
- 128 bytes of on-chip data memory.
- 64 kilobytes of external program and RAM addressability.
- Four register banks.
- 32 bidirectional I/O lines organized as four 8-bit ports.

Other members of the MCS-51 microcontrollers might be emulated in the
future as they are very similar.
